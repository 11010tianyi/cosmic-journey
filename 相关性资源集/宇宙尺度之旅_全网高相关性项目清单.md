# 宇宙尺度之旅（Cosmic Journey）— 全网高相关性项目集合

> **说明**：本文档以本仓库「交互式宇宙尺度 / 对数缩放 / 时空漫游」定位为锚点，汇总主题高度相关的网页、应用与科研向资源，并补充同类高相关条目。  
> **排序规则**：按 **与本项目主题的相关性得分（10 分制，高→低）** 降序排列；同分按「网页可即时体验 → 桌面/客户端 → 科研基础设施」大致分组。

**相关性打分维度（综合）**  
1. 是否核心呈现 **多数量级尺度** 或 **时空连续漫游**（与哲学文档中的对数映射、Cosmic Zoom 最接近）；  
2. 是否 **3D 可飞行 / 轨道级物理或数据驱动**；  
3. 是否 **权威天文数据 / 科普或科研背书**；  
4. 与「从星系到地球 / 人类文明尺度」叙事链路的贴合度。

## 重点条目：官方入口与产品形态（Universe Sandbox / NASA Eyes / Gaia Sky）

以下为 **Universe Sandbox 官方形态** 与 **NASA Eyes / Gaia Sky** 的补充说明，已反映在下文总表「链接直达」列与分项链接表中。

### Universe Sandbox（物理宇宙沙盒）

- **官方网站**：https://universesandbox.com/  
- **产品形态**：以 **Windows / macOS / Linux 可下载客户端** 为主（常见分发渠道含 Steam），**目前无与桌面版对等的官方纯网页版**。  
- **常见误链**：网络上出现的 `https://universe.sandbox.game/` **并非** Universe Sandbox 模拟器官方入口，易与同名或元宇宙类产品混淆；请以 **universesandbox.com** 为准。  
- **非官方网页克隆**：存在爱好者用 WebGL/JS 制作的「重力玩具」级页面或开源仓库，**不等同**于官方产品在物理模型、气候/天体事件等方面的完整度，仅可作轻量体验参考。

### NASA Eyes 系列（浏览器 3D + 任务叙事）

- **总入口（Science @ NASA）**：https://science.nasa.gov/eyes/  
- **Eyes on the Solar System**（太阳系 3D、航天器轨迹、时间轴）：https://eyes.nasa.gov/apps/solar-system/  
- **Eyes on Exoplanets**（系外行星 3D 浏览）：https://eyes.nasa.gov/apps/exo/  
- **系外行星专题说明页（可交叉核验）**：https://science.nasa.gov/exoplanets/eyes-on-exoplanets-web/  
- **背景报道示例**（新版网页 Eyes 与太阳系探索）：https://www.nasa.gov/solar-system/explore-the-solar-system-with-nasas-new-and-improved-3d-eyes/  
- 同系列通常还包括 **Eyes on Asteroids**、**Eyes on the Earth** 等，适合与 **ViewSpace**、**NASA Solar System Exploration** 叠用。

### Gaia Sky（Gaia 星表 3D 浏览）

- **项目下载（常用聚合页）**：https://gaiasky.space/downloads/  
- **海德堡大学 ZAH Gaia Sky 下载页**：https://www.zah.uni-heidelberg.de/gaia/outreach/gaiasky/downloads/  
- **文档门户（版本号随更新变化）**：https://gaia.ari.uni-heidelberg.de/gaiasky/docs/  
- **源代码**：https://codeberg.org/gaiasky/gaiasky

## 扩展条目补充说明（为何入选）

- **Scale of the Universe 3.0 / 开源重制**：与 htwins 经典体验同题，技术实现与可维护性更贴近当代 Web 开发，适合与「本仓库 p5.js / 网页交互」对照。  
- **Interactive Cosmic Scale Tool**：同题滚动缩放 + 分层解说，适合作为竞品/教学设计参考。  
- **Stellarium Web / NASA Eyes / Gaia Sky / WWT**：尺度叙事弱于「powers-of-ten」类页面，但在 **真实数据驱动的 3D 天球与任务可视化** 上与「从地球望向深空」一脉相承；其中 **NASA Eyes** 的太阳系与系外行星应用以 **浏览器即开** 为主（URL 见总表）。  
- **Mitaka / SpaceEngine**：前者偏数值宇宙学展示，后者偏全宇宙引擎漫游，均与 OpenSpace、Celestia 同属「深空沉浸」簇。  
- **千衍**：与「数字宇宙」叙事一致，强调真实物理与算力；链接列为国家天文台公开报道入口，便于了解项目背景。

## 使用建议

- **做尺度交互与 UI/UX**：优先对标表中 **9–10 分** 条目 + 本仓库 `cosmic_journey_philosophy.md` 中的对数映射与层级视觉语言。  
- **做数据真实感与任务叙事**：叠用 **OpenSpace、NASA 系、Gaia Sky、Stellarium Web**。  
- **做时间维与文明尺度**：**ChronoZoom、Deep Time Walk** 与 **Lightyear.fm** 分别对应「宇宙史」「地质史」「电磁文化史」三条轴。
